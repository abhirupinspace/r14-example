# Root14 MCP Examples

Walkthroughs for using Root14 via MCP (Model Context Protocol) tools in Claude Code or any MCP-compatible client.

## Prerequisites

- Root14 MCP server configured in your Claude Code settings
- Stellar testnet account with funds

## Walkthroughs

| Guide | Description |
|-------|-------------|
| `01-setup-wallet.md` | Generate keypair and configure wallet |
| `02-deposit-and-balance.md` | Deposit funds and check balance |
| `03-private-transfer.md` | Private transfer with ZK proofs |
